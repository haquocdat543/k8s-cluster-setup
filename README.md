## ON PREM 

### VAGRANT 

#### Prerequisites
##### Virtualbox installed
##### Vagrant installed
#### Use following command to setup k8s cluster on-prem
```
cd k8s-cluster-setup/on-prem/vagrant
vagrant up
```
#### Use following command to destroy k8s cluster on-prem
```
cd k8s-cluster-setup/on-prem/vagrant
vagrant destroy
```
